# Inkraft
使用[ink语言](https://github.com/inkle/ink)为Minecraft编写交互式对话。

## 功能特性

- 在游戏中，引擎全部使用命令的方式交互，因此可以与其他模组一起工作
- 可以以仅服务端模式运行，无需客户端安装
- 提供了大量外部函数可在ink语言中调用
  - 与玩家物品栏交互
  - 获取游戏世界时间
  - 执行命令
  - 操作计分板和命令存储
  - 获取真实世界时间
- 提供了全局变量机制用于在不同ink脚本之间存储信息
