# Inkraft

Write interactive conversations for Minecraft with the [Ink script language](https://github.com/inkle/ink).

Other languages: [简体中文](README.zh.md)

## Features

- In game, the engine is entirely command-driven, so it can work together with other mods.
- It can run in server-only mode, without requiring clients to install it.
- It provides many external functions that can be called from Ink scripts:
  - Interact with player inventories.
  - Get in-game world time.
  - Execute commands.
  - Operate on scoreboards and command storage.
  - Get real-world time.
- It provides a global variable mechanism for storing information across different Ink scripts.
